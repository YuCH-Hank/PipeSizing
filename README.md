# 風量管路繪製工具

瀏覽器端的繪圖工具：右鍵新增節點、距離推算線段、依統一風速給出管徑建議並顯示效率，附除錯 log 視窗。

## 主要功能
- 右鍵開啟自訂選單，可新增進風口 / 節點 / 出風口；節點可拖曳。
- Shift+拖曳（或中鍵拖曳）可平移整個畫布，行動裝置可拖曳節點與查看 log。
- 連線模式按兩個節點建立管路，長度依兩點距離（px/m 比例）計算，與點擊順序無關。
- 全域風速預設 13 m/s，調整後自動套用至所有管路。
- 建議管徑以 50 mm 級距進位，效率超過 90% 再加 50 mm 重新計算；圓管效率公式 `(理論/推薦)^2*100`，方管效率 `理論截面積/實際截面積*100`。
- 線段設定先選管型，再展開其他欄位；標籤改為縱向排列顯示長度 / 尺寸 / 效率，線粗與建議尺寸成正比。
- 節點與線段表單採左右排列，介面文案為繁體中文；左下有 #debugLog 便於除錯。

## 使用方式
1. 直接以瀏覽器開啟 `index.html`（桌機或手機皆可）。
2. 右鍵畫布新增節點，拖曳調整位置；Shift+拖曳可平移畫布。
3. 按「連線模式」後依序點兩個節點建立管路；比例(px/m) 會影響長度計算。
4. 左側可調整預設風速、管型與比例；選取節點可編輯風量 CMM；選取線段可設定管型、方管寬高。
5. 「重新計算」會重跑風量累積與管徑建議；「匯出結果」可複製文字摘要；「清除全部」重置畫布。

## 檔案結構
- `index.html`：頁面結構與按鈕配置（含連線模式、切換網格）。
- `src/styles.css`：樣式與佈局（含左右表單與 log 面板）。
- `src/main.js`：互動邏輯、風量累積、管徑計算與除錯紀錄（繁中 Doxygen 註解）。
